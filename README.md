# InitForge

A modular Python CLI tool to help initialization of projects.
## Features

- Flask app structure generation
- MongoDB integration via MongoEngine
- Twitter API setup via Tweepy
- OpenAI integration
- Modular and extensible architecture

## Installation

```bash
pip install initforge
```

## Development Setup

```bash
git clone git@github.com:czuger/initforge.git
cd initforge
pip install -e ".[dev]"
```

## Running Tests

```bash
pytest tests/
```

## Requirements

- Python 3.13+
